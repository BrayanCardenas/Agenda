# Agenda de Contactos

Aplicación web sencilla para gestionar contactos (nombre, número y dirección).  
Permite **añadir, listar y eliminar** contactos usando `localStorage` como base de datos en el navegador.

---

## Características
- Formulario para agregar contactos con validación básica.
- Renderizado dinámico de la lista de contactos.
- Eliminación de contactos con un solo clic.
- Persistencia de datos en `localStorage` (los contactos se mantienen aunque cierres el navegador).
- Interfaz minimalista con íconos de Google Fonts.

---

## Tecnologías utilizadas
- **HTML5** para la estructura.
- **CSS3** para estilos básicos.
- **JavaScript (ES6)** para la lógica de la aplicación.
- **localStorage** para persistencia de datos.
- **Google Fonts** (Poppins y Material Symbols).
